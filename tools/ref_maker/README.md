# ref 이미지 제작기

상세지도에서 **고정 크기**로 크롭해 보물지도 ref PNG를 만드는 독립 도구입니다.

## 실행

프로젝트 루트에서:

```bash
python tools/ref_maker/main.py
```

## 기능

- `zones.json` 지역 선택 → `assets/maps/{확장팩}/{zone_id}.png` 자동 로드
- **줌** (휠 / 슬라이더): 인게임 200%에 맞추려면 기본값 `2.0` 사용
- **크롭 크기**: 기본 `218×188` (기존 ref 호환). **인게임 창 비율 501:430** 고정 옵션
- **줌 → 크롭 크기 맞춤**: 현재 줌에 맞는 크롭 자동 계산 (줌 2.0 → 약 250×215)
- **게임 좌표 입력**: `22.61, 15.62` 붙여넣기 또는 X/Y 입력 → **마커 배치**
- **보물 마커**: 노란 크롭 프레임 **안**을 클릭 → `assets/treasure_marker.png` 배치 + 게임 좌표 표시
- **PNG 저장**: `assets/treasure_refs/{확장팩}/{zone_id}/solo|party8/{x}_{y}.png` 경로 안내

## ref 제작 팁

1. 인게임 보물지도와 비슷한 줌(보통 2.0)으로 맞춘 뒤 크롭
2. 보물 마커는 인게임 X 위치 근처에 배치 (매칭·상세지도 표시에 유리)
3. 저장 후 좌표 DB 갱신:

```bash
python scripts/build_treasure_ref_coords.py <zone_id>
```

## 조작

| 입력 | 동작 |
|------|------|
| 휠 | 줌 인/아웃 |
| 크롭 밖 드래그 | 지도 이동 |
| 크롭 안 클릭 | 보물 마커 배치 + 좌표 |
| 방향키 | 미세 이동 |
| Shift+방향키 | 빠른 이동 |
